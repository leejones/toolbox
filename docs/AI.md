## Perspectives

### Blog Post: AI DEMANDS MORE ENGINEERING DISCIPLINE. NOT LESS

https://charity.wtf/2026/06/15/ai-demands-more-engineering-discipline-not-less-xpost/
by Charity Majors

> The real product of a software team is shared understanding.

- However, production (reality) is the true product regardless of how we understanding it.
- Sounds like John Allspaw's above the line below the line distinction.  Above the line is our shared (or unshared) understanding of the system and below the line is the actual system.

> AI pushes this premise beyond infrastructure and into application code itself. When rewriting is cheap, editing in place becomes risky. Mutation accumulates entropy. Replacement resets it.

- That's a quote from Chad Fowler that Charity Majors references.
- Code used to be reality, and now that AI makes it easier to change, it's more of a cached version of our current understanding.  We can change it to match our new understanding.

> The shift from handcrafted servers to immutable infrastructure taught us that mutability is the sworn enemy of understanding. Any artifact that is edited in place creates drift. Drift is what makes systems impossible to maintain.

- If we can delete our code and confidently recreate it, it shows that we don't understand it.  If we don't know what will break when we delete it, we don't understand it.
- Move the understanding from our heads to docs and arch diagrams.  The code because a materialized view of our understanding.
- Fast feedback loops are more important than ever.

### Blog Post: A frontier without an ecosystem is not stable

https://snscratchpad.com/posts/frontier-ecosystem/
by Satya Nadella

- Token capital is essential, but not sufficient. Human capital is what will differential organizations.
- An organization's learning loop is how it uses AI to enhance it's human capabilities.
- AI makes it easy for companies to absorb new information, but doesn't necessarily allow them to leverage it.
- Without care, AI could hollow out the economy so that it looks ok on the surface but people are suffering.
