# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## About This Repository

A personal knowledge base ("outboard brain") of Markdown reference documents covering tools, commands, and concepts. All content lives in `docs/` as standalone `.md` files.

## Content Conventions

- Files start with `## H2` section headings — no top-level `# H1` title per file (the filename serves as the title)
- Command examples use fenced code blocks; include language hints (` ```bash `) when relevant
- Cite sources with a `-- URL` line or `Source: URL` after the relevant section
- Use `<placeholder>` syntax for user-supplied values in command examples (e.g., `<namespaceName>`)
- Ordered lists with embedded code blocks require double-indentation (4 spaces) to render correctly in Markdown
